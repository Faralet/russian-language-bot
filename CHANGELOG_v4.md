# Changelog v4 - «вкусный» тренажер ЕГЭ/ОГЭ

Крупное продуктовое обновление поверх v3.3. Цель: убрать «ИИ-налет», сделать
бота полезным и результативным для ЕГЭ/ОГЭ и удобным в ежедневном использовании.
Все миграции (новые поля/таблицы) применяются автоматически при старте.

## Контент по ФИПИ
- Банк расширен примерно до **876 заданий** (было ~500): добавлено ~376 оригинальных
  заданий в формате ФИПИ (11 пакетов: `seed_content_ege`, `_orth`, `_input`,
  `_text`, `_input_ext`, `_depth`, `_pack2..6`).
- Покрытие по номерам ЕГЭ: №4 ~59, №5 ~40, №6 ~24, №7 ~32, №8 ~18,
  орфография №9-15 ~70, пунктуация №16-21 ~30, текстовые №1-3/23-26 - по 4-7.
- Вычищен шаблонный наполнитель из старого контента (повторяющиеся «факты» и приписки).
- Тон переписан под живого методиста (обращение на «ты»), без ИИ-поэтики.

## Формат заданий
- Новый тип `text_input` - ввод ответа с клавиатуры (как на реальном ЕГЭ),
  интегрирован в основной поток занятий (идет в статистику и интервальное повторение).
- Нормализация ответа (регистр, е/ё, пробелы).

## Геймификация и удержание (по практикам Duolingo)
- Урок из нескольких этапов (`LESSON_STAGES`), а не 5 фиксированных вопросов.
- Серии с заморозкой (streak freeze), празднование рубежей 7/14/30/...
- Дневная цель (`DAILY_GOAL_QUESTIONS`) с прогресс-баром.
- Достижения (10 бейджей) и уровни (XP = верные ответы).
- Прогресс в баллах: ориентир по официальной шкале ФИПИ; экран «Мой балл» -
  готовность по каждому заданию ЕГЭ и слабые места.

## Новые разделы
- «Сочинение №27»: структура, критерии К1-К10, клише, банк проблем, проверка
  объема (>=150 слов), AI-проверка по К1-К10 (за флагом `ENABLE_AI`).
- «ОГЭ»: изложение (приемы сжатия, ИК1-ИК3), сочинение 9.1/9.2/9.3, проверка объема.
- «Текст с заданиями» (passage-блок): 7 текстов с заданиями на тип речи,
  средства связи и выразительности.
- Диагностика: стартовый тест из разных тем с ориентиром по баллу.
- Путь обучения: темы по порядку с привязкой к номерам ЕГЭ и прогрессом.

## Рост
- Рефералка «приведи друга» (deep-link, бонус обоим).
- Кнопка «Поделиться результатом».

## Новые переменные окружения
`LESSON_STAGES`, `DAILY_GOAL_QUESTIONS`, `OPENAI_BASE_URL` (см. `.env.example`).

## Новые поля/таблицы БД (миграции автоматические)
- `users`: `referred_by`, `referral_count`, `current_streak`, `longest_streak`,
  `streak_freezes`, `last_activity_on`.
- Новая таблица `user_achievements`.

## Проверка
- `compileall` всего `app/` - без ошибок.
- Полный импорт `app.main` со всеми хендлерами - без ошибок (aiogram 3.29).
- ORM-модели загружаются; банк (~876) и тексты (7) проходят валидацию (0 ошибок, 0 дублей).
